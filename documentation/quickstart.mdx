## Basic Request

Let's check for current environmental risks at Half Dome in Yosemite:

<CodeGroup>
```bash cURL
curl "https://api.goes.health/v1/risks?location=37.7325,-119.5581&api_key=YOUR_API_KEY"
```

```jsx React
import { useEffect, useState } from 'react';

function RiskCheck() {
  const [risks, setRisks] = useState([]);

  useEffect(() => {
    fetch(`https://api.goes.health/v1/risks?location=37.7325,-119.5581&api_key=${process.env.REACT_APP_GOES_API_KEY}`)
      .then(res => res.json())
      .then(data => setRisks(data.risks));
  }, []);

  return (
    <div>
      {risks.map(risk => (
        <div key={risk.id}>
          <h3>{risk.title}</h3>
          <p>Severity: {risk.severity}</p>
        </div>
      ))}
    </div>
  );
}
```

```jsx React Native
import React, { useEffect, useState } from 'react';
import { View, Text, FlatList } from 'react-native';

function RiskCheck() {
  const [risks, setRisks] = useState([]);

    fetch(`https://api.goes.health/v1/risks?location=37.7325,-119.5581&api_key=${process.env.GOES_API_KEY}`)
      .then(res => res.json())
      .then(data => setRisks(data.risks));
  }, []);

  return (
    <FlatList
      data={risks}
      keyExtractor={item => item.id}
      renderItem={({ item }) => (
        <View style={{ padding: 10 }}>
          <Text>{item.title}</Text>
          <Text>Severity: {item.severity}</Text>
        </View>
      )}
    />
  );
}
```

```js Node.js
const GOES_API_KEY = process.env.GOES_API_KEY;

async function checkRisks() {
  const response = await fetch(
    `https://api.goes.health/v1/risks?location=37.7325,-119.5581&api_key=${GOES_API_KEY}`
  );
  
  if (!response.ok) {
    throw new Error(`API error: ${response.status}`);
  }
  
  const data = await response.json();
  console.log(data.risks);
}
```

```python Python
import os
import requests
from typing import Dict, List

def check_risks() -> Dict:
    response = requests.get(
        "https://api.goes.health/v1/risks",
        params={
            "location": "37.7325,-119.5581",
            "api_key": os.environ["GOES_API_KEY"],
        },
        timeout=10,
    )
    response.raise_for_status()
    return response.json()
```
</CodeGroup>

## Precise Request

Request specific risks by passing their IDs. Available risk IDs: `sunburn`, `high-altitude`, `heat-illness`, `breathing-problems`, `frostbite`, `hypothermia`, `lightning`.

<CodeGroup>
```bash cURL
curl "https://api.goes.health/v1/risks?location=37.7325,-119.5581&ids=sunburn,high-altitude&api_key=YOUR_API_KEY"
```

```jsx React
function RiskCheck() {
  const [risks, setRisks] = useState([]);
  
  useEffect(() => {
    fetch(`https://api.goes.health/v1/risks?location=37.7325,-119.5581&ids=sunburn,high-altitude&api_key=${process.env.REACT_APP_GOES_API_KEY}`)
      .then(res => res.json())
      .then(data => setRisks(data.risks));
  }, []);

  return <div>{risks.map(risk => <div key={risk.id}>{risk.title}</div>)}</div>;
}
```

```python Python
def check_specific_risks():
    response = requests.get(
        "https://api.goes.health/v1/risks",
        params={
            "location": "37.7325,-119.5581",
            "ids": "sunburn,high-altitude",
            "api_key": os.environ["GOES_API_KEY"],
        },
    )
    return response.json()
```
</CodeGroup>

This behavior is also supported by the [/wildlife](/api-reference/wildlife) endpoint. 

## Complete Request

Get a comprehensive overview including all risks `sunburn`, `high-altitude`, `heat-illness`, `breathing-problems`, `frostbite`, `hypothermia`, `lightning`, `poison-sumac`, `poison-oak`, `poison-ivy`, `bears`, `venomous-snakes`, `mosquitoes`, `ticks`, national alerts, and a personalized AI summary.

<CodeGroup>
```bash cURL
curl "https://api.goes.health/v1/overview?location=37.7325,-119.5581&activity=hiking&api_key=YOUR_API_KEY"
```

```jsx React
function CompleteOverview() {
  const [data, setData] = useState({});
  
  useEffect(() => {
    fetch(`https://api.goes.health/v1/overview?location=37.7325,-119.5581&activity=hiking&api_key=${process.env.REACT_APP_GOES_API_KEY}`)
      .then(res => res.json())
      .then(setData);
  }, []);

  return (
    <div>
      <p>{data.summary}</p>
      {data.risks?.map(risk => <div key={risk.id}>{risk.title}</div>)}
    </div>
  );
}
```

```python Python
def get_complete_overview():
    response = requests.get(
        "https://api.goes.health/v1/overview",
        params={
            "location": "37.7325,-119.5581",
            "activity": "hiking",
            "api_key": os.environ["GOES_API_KEY"],
        },
    )
    return response.json()
```
</CodeGroup>
